Representación del TDA MaxQueue

Invariante de la representación

  • MaxQueue almacena elementos de tipo element.
  • Cada element contiene un valor int y el máximo int de la cola hasta su inserción.
  • element.max_value >= element.value

Función de abstracción

Un objeto válido de MaxQueue representa una cola que permite manipular sus elementos y consultar en cualquier momento el valor máximo.
