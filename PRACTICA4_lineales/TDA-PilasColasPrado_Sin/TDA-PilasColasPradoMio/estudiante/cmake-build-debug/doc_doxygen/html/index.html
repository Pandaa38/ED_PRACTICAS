¿Qué es el TDA MaxStack?

Nuestro TDA MaxStack es una instancia que representa una pila que permite realizar las operaciones básicas que porporciona una pila, , además de consultar el valor máximo actual alojado en la pila.

¿Qué es el TDA MaxQueue?

Nuestro TDA MaxQueue es una instancia que representa una cola que permite realizar las operaciones básicas que porporciona una cola, , además de consultar el valor máximo actual alojado en la cola.

MaxStack:

Se trata de una pila con capacidad para almacenar y obtener el máximo elemento en cualquier momento.

‍pila_max < >

Los argumentos permitidos son carcateres que sean números y puntos. Notemos que no se puede introducir un solo punto.

Parameters
<Numeros>Numeros entero que se va a insertar
<Puntos>Cada punto indica un entero a extraer
